§ Конспект урока №5. Поля, отступы, box-sizing.

1. Свойство box-sizing

Описание: В зависимости от выбранного значения свойства, браузер по-разному будет рассчитывать значения свойств width/height примененных к блочным элементам.

Блок с красной границей и блок с зеленой границей, расположенные ниже в столбец, имеют следующие одинаковые значения свойств:

margin: 20px;
padding: 20px;
border: 2px solid some-color;
max-width: 500px;

Как мы видим, при достаточной ширине элемента-родителя обоих блоков их ширина должна быть 500px, как мы это и прописали в стилях. Но получается следующее поведение.

К блоку с красной границей применено значение content-box. Это означает, что помимо основной ширины, явно прописанной в стилях, к ней будут добавлены значения внутренних отступов (padding) справа и слева и границ (border) тоже справа и слева. И в итоге ширина блока будет составлять 544px.

500px + 20px + 20px + 2px + 2px = 544px

Записать свойство padding можно несколькими способами.

  1. padding: 10px 20px 30px 40px; - запись означает: сверху будет поле размером 10px, справа - 20px, снизу - 30px, слева - 40px;
  2. padding: 10px 20px 30рх; - сверху будет 10px, влева/справа - 20рх; снизу - 30рх
  3. padding: 10px 20px; - сверху/снизу будет 10px, влева/справа будет 20рх;
  4. padding: 10px; - задаст поля размером 10рх сразу со всех сторон.

Для простоты понимания того, в какой последовательности будут применены значения свойств из первого пункта списка, справа приводится рисунок.

Также существует возможность задать значение полей отдельно для каждой стороны.

  1. padding-top - задаст размер поля сверху
  2. padding-right - справа
  3. padding-bottom - снизу
  4. padding-left - слева

Запись свойства margin полностью аналогична записи свойства padding.

1. Поля (padding) и отступы (margin) не применимы к строчным элементам!
2. При указании полей и отступов в процентах, значение считается от ширины элемента-родителя!

4. Свойство width

Описание: устанавливает ширину для всех блочных элементов и для строчного элемента <img>.

Также существуют "более гибкие" аналоги свойства width, это min-width и max-width.

3.1 min-width устанавливает минимальную ширину блочных элементов и элемента <img>. Это означает, что блок не может принимать ширину меньше, чем указано в значении свойства, а максимальное значение ограничено шириной родительского элемента.

3.2 max-width устанавливает максимальную ширину блочных элементов и элемента <img>. Это означает, что элемент не может быть больше указанного в свойстве значения, а минимальный размер ограничивается содержимым элемента.

Пример №1. К зелёному блоку применено значение min-width: 500px;

Ширина зеленого блока будет равна 50% ширины черного блока, но не меньше красной границы, т.е. 500px.

Пример №2. К зелёному блоку применено значение max-width: 500px;

Максимальная ширина зеленого блока будет равна 500рх, при уменьшении ширины экрана браузера - 50% от ширины черного блока.
